## Installation

```bash
$ yarn install
```

## create .env file
You need to create .env file in the root of the project, then put the environment variables in the file. You can be guide by the .env.example file

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
